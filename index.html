Directions

Find the mod you want, then click on the "Download" button next to it
Then, go to Sonic 3 AIR main menu, then select "Exit"
Then, on the next screen there will be a few folders, enter the mods folder
Then click the import button in the top right (the up arrow) and select the zip file of the mod you want to install
Then click "Restart Game"
Then once you are on the main menu again, then select "Mods"
Then in the mod menu use the arrow keys to go to the mod you want to enable, hold A while hovered over the mod you want to enable, then press left arrow then hit enter
If you want to disable a mod then hold A on the mod you want to disable, and press right arrow
